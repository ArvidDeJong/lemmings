# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an email to Arvid de Jong at arvid@darvis.nl.

All security vulnerabilities will be promptly addressed.

Please do not publicly disclose the issue until it has been addressed by the maintainers.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
